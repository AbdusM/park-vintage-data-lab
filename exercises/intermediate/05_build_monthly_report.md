# Mission 6: Build a monthly report

## Goal

Create a simple business report in markdown.

## Tasks

1. Use `notebooks/05_simple_etl_pipeline.ipynb` to generate a monthly summary.
2. Open `reports/my_first_park_vintage_report.md`.
3. Fill each required section:

- Top selling categories
- Best sales channel
- Highest margin items
- Slow-moving inventory
- Recommendations

## Acceptance criteria

- Report has at least 3 findings.
- Report has at least 2 recommendations.
