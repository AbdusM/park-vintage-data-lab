# Mission 4: Run your first SQL query

## Goal

Run a SQL query to list in-stock inventory.

## Tasks

1. Build the SQLite database by running `python src/build_store_database.py`.
2. Open `sql/01_select_inventory.sql`.
3. Run the query from SQL notebook or a local SQLite client.

## Acceptance criteria

- You can explain `SELECT`, `WHERE`, and `ORDER BY`.
- You can list in-stock items from SQL.
