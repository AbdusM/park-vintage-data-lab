# Mission 1: Learn the terminal

## Goal

Find where the raw inventory data lives and understand project folders.

## Tasks

1. From the repo root, run `pwd`.
2. Run `ls`.
3. Move into the raw data folder with `cd data/raw`.
4. Run `ls` and locate `inventory.csv`.
5. Return to root with `cd ../..`.

## Acceptance criteria

- You can explain what `data/raw/` means.
- You can locate `data/raw/inventory.csv`.
