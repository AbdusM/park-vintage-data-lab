# AGENTS.md

## Project purpose

Park Vintage Data Lab is a beginner-friendly learning repo for CLI tools, Python, Jupyter notebooks, SQL, GitHub, and Codex.

## Style rules

- Keep code simple.
- Prefer readability over cleverness.
- Use beginner-friendly comments.
- Avoid adding heavy frameworks.
- Do not introduce Docker, Airflow, dbt, or cloud tools in v1.

## Data rules

- Never overwrite files in `data/raw/`.
- Write cleaned files to `data/processed/`.
- Keep all data fictional.
- Do not add personal or sensitive data.

## Good assistant tasks

- Explain a file.
- Add comments.
- Create a small exercise.
- Add a simple SQL query.
- Improve README clarity.
- Add tests for beginner scripts.

## Bad assistant tasks

- Rebuild the whole repo.
- Add unnecessary frameworks.
- Replace beginner code with advanced abstractions.
- Delete raw data.
